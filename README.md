## 📌 Descripción

**Hidrología Aplicada** es un proyecto técnico enfocado en el análisis hidrológico mediante herramientas computacionales y métodos de ingeniería.

Incluye el estudio y aplicación de:

- 🌧️ Precipitación y escurrimiento
- ⛰️ Análisis de cuencas hidrológicas
- ⏱️ Tiempo de concentración (Tc)
- 📊 Métodos como Kirpich, SCS y otros
- 🐍 Programación aplicada en Python
- 📐 Modelos hidráulicos básicos

---

## 🚀 Objetivo

Facilitar el aprendizaje de la hidrología aplicada mediante:

- Explicaciones claras
- Ejemplos prácticos
- Cálculos automatizados
- Herramientas en Python y Excel
- Aplicación en proyectos reales de ingeniería

---

## 🧰 Herramientas utilizadas

- Python 🐍  
- Excel 📊

## 📂 Estructura del proyecto



🌊 Descripción del blog

Hidrología Aplicada es un blog técnico enfocado en la ingeniería hidrológica, donde se comparten conceptos, métodos de cálculo, herramientas y aplicaciones prácticas para el análisis del agua en cuencas, escurrimientos, precipitación y modelación hidrológica.

El objetivo es apoyar el aprendizaje y la práctica profesional mediante contenido claro, ejemplos reales y desarrollo de herramientas en Python, Excel y software especializado.

Este espacio está diseñado para estudiantes, ingenieros y profesionales interesados en comprender y aplicar la hidrología de forma práctica en proyectos reales.
